SENTINEL
Solo Developer — Production Data Pipeline
Live · AWS

Every day, the U.S. government publishes data that moves markets — interest rate decisions, inflation numbers, unemployment reports. Traders watch it. Most people miss it entirely.

SENTINEL watches it automatically. It pulls real economic data from the Federal Reserve's database every morning, stores it, and flags when something unusual happens — a spike in inflation, a jump in unemployment, an unexpected rate shift. No news article to read. No spreadsheet to update. Just an alert when the numbers move.

The system runs 24/7 on a cloud server. Before going live, I audited the entire codebase and resolved 6 security vulnerabilities — including one that could have let an outsider delete the database. The infrastructure is defined in code: if the server ever goes down, one command rebuilds the whole thing from scratch.

About Me

I studied Finance at USF and didn't write a single line of code until my final semester — then one Computer Science class rewired how I think. I taught myself Python, started automating everything I could, and never looked back.

That pivot took me from Wells Fargo to Raymond James to Citibank, where I now build automation infrastructure for the Operational Risk team. Along the way I created ATLAS — a Python-based platform I designed and built from scratch as the sole developer. Outside of Citi, I've built SENTINEL — a live financial data pipeline on AWS that ingests real economic indicators, detects anomalies, and serves alerts through a secured FastAPI layer. And ThriveKid — a React + C# PWA on Vercel and Azure. Both are fully deployed and publicly shareable.
